# Measurement Uncertainty Evaluation Script

This repository provides a Python script for evaluating measurement uncertainty in the determination of total nitrogen (%N) in ZA fertilizer via formol titration, following the ISO-GUM framework.

## Description
The script performs:
- NaOH standardization
- Nitrogen content (%N) calculation
- Repeatability analysis
- Model-based uncertainty propagation
- Combined uncertainty estimation
- Contribution analysis of uncertainty sources

## Requirements
- Python (no external dependencies required)

## How to Run
```bash
python measurement_uncertainty.py
